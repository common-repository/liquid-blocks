== Description ==

Design your site with very simple yet powerful blocks of slider and accordion blocks, and streamline editing with beautiful patterns and extended toolbars.

In Gutenberg Block Template & Design Gallery, Just select your favorite design. Instantly create page section elements. Like a very simple page builder.

You can manage Block Pattern. (WordPress 5.5+)
Fully compatible with Gutenberg and themes and AMP.
latest information on [LIQUID PRESS](https://lqd.jp/wp/).

https://www.youtube.com/watch?v=pNpv_SA_NNY

= Blocks =

Simple yet powerful blocks.
* Carousel Slider (Slideshow, Fade)
* Accordion
* Card
* Custom Field

Add to RichText Toolbar.

* Underline

* Small

= Patterns =

Carefully selected 37+ Free Designs.

* Price list
* CTA
* Landing pages
* Block Patterns (You can register.)

== Installation ==

1. Upload 'liquid-blocks' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use 'LIQUID BLOCKS' in the 'Gutenberg editor' menu

== Frequently Asked Questions ==

= Q: This block contains unexpected or invalid content? =

A: Try More options > Attempt Block Recovery.
